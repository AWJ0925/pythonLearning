### 3. 运算符

| python表达式                 | Result             | description          |
| ---------------------------- | ------------------ | -------------------- |
| len([1, 2, 3])               | 3                  | 计算序列中元素的个数 |
| [1, 2, 3] + [4, 5, 6]        | [1, 2, 3, 4, 5, 6] | 组合                 |
| ['py'] * 2                   | ['py', 'py']       | 元素的赋值           |
| 3 in [1, 2, 3]               | True               | 元素是否存在于序列中 |
| 3 not in [1, 2, 3]           | False              | 元素是否不在序列中   |
| for e in [1, 2, 3]: print(e) | 1 2 3              | 迭代                 |
